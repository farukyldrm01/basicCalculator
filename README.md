# Hesap Makinesi

Basit bir iOS hesap makinesi uygulaması.

## Özellikler

- Toplama
- Çıkarma
- Çarpma
- Bölme

## Gereksinimler

- Xcode
- iOS 13.0+

## Kurulum

1. Projeyi Xcode ile açın
2. Simulator veya gerçek cihazda çalıştırın

## Kullanım

İki sayı girin ve işlem butonlarından birine tıklayın.
